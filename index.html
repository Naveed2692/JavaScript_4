5 Legal variables

  1. A variable name can't contain any spaces. like (var userName = "Naveed";)
  2. A variable name can contain only letters, numbers, dollar signs, and underscores.
    Like (var user_name = "Naveed";)
  3. Though a variable name can't be any of JavaScript's keywords, it can contain keywords.
    For example, userAlert and myVar are legal
  4. Variable name are case sensitive.
    If you assign variable "name" with string "Naveed" and ask to JavaScript "Name" Result will be empty
  5. To make your variable more readable you have to us camelCase.
    Like (var capitalCity = "Karachi";)

5 Illigal variables

  1. A variable name contain any spaces. like (var user Name = "Naveed";)
  2. A variable name can't contain only letters, numbers, dollar signs, and underscores.
    Like (var !user_name = "Naveed";)

“Rules for naming JS variables”

Variable names can only contain letter, number, $signs and underscores. For example $my_1stVariable
Variables must begin with a number, $signs or underscores. For example $name, _name or name
Variable names are case sensitive
Variable names should not be JS keywords
